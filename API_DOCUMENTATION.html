📚 Blog REST API Documentation

Base URL

http://localhost:1820/api/v1

Authentication

All protected endpoints require a JWT token in the header:

Authorization: Bearer <token>

Authentication Endpoints

POST/auth/register

Description: Create a new user account

Body:

{ "username": "jean", "email": "jean@example.com", "password": "password123" }

Response (200):

{ "token": "eyJhbGciOiJIUzUxMiJ9...", "username": "jean", "message": "Registered successfully" }

POST/auth/login

Description: Authenticate and get JWT token

Body:

{ "username": "jean", "password": "password123" }

Response (200):

{ "token": "eyJhbGciOiJIUzUxMiJ9...", "username": "jean", "message": "Login successful" }

Post Endpoints

POST/posts

Authorization: Bearer <token>

Body:

{ "title": "My First Post", "content": "This is the content" }

GET/posts

Description: List all posts

Response (200): Array of posts

GET/posts/{id}

Description: Get a single post

PUT/posts/{id}

Authorization: Bearer <token>

Body:

{ "title": "Updated Title", "content": "Updated content" }

DELETE/posts/{id}

Authorization: Bearer <token>

Comments Endpoints

POST/comments

Authorization: Bearer <token>

Body:

{ "postId": 1, "content": "Great post!" }

GET/comments?postId=1

Description: List all comments for a post

DELETE/comments/{id}

Authorization: Bearer <token>

Ratings Endpoints

POST/ratings

Authorization: Bearer <token>

Body:

{ "postId": 1, "score": 5, "comment": "Excellent post!" }

GET/ratings?postId=1

Description: Get all ratings for a post

DELETE/ratings/{id}

Authorization: Bearer <token>

Status Codes

Code Meaning
200 Success
201 Created
400 Bad Request
401 Unauthorized
403 Forbidden
404 Not Found
500 Server Error

Example Usage with cURL

1. Register

curl -X POST http://localhost:1820/api/v1/auth/register \ -H "Content-Type: application/json" \ -d '{"username":"jean","email":"jean@example.com","password":"pass123"}'

2. Login

curl -X POST http://localhost:1820/api/v1/auth/login \ -H "Content-Type: application/json" \ -d '{"username":"jean","password":"pass123"}'

3. Create Post (with token)

TOKEN="eyJhbGciOiJIUzUxMiJ9..." curl -X POST http://localhost:1820/api/v1/posts \ -H "Content-Type: application/json" \ -H "Authorization: Bearer $TOKEN" \ -d '{"title":"My Post","content":"Content here"}'

4. List Posts

curl http://localhost:1820/api/v1/posts
